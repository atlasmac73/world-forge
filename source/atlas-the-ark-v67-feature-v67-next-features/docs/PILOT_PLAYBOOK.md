## THE PILOT PRINCIPLE

> "Do not test the entire empire. Test the smallest workflow that proves the empire can exist."

**First pilot objective:** Use Atlas to manage one real contractor/project workflow from scattered data to organized action.

---

## PILOT SCOPE

**Includes:**
- 1 workspace, 1-3 users, 1-5 projects
- Google connector (read-only, selective import)
- Manual file upload
- Project Kanban (15 columns)
- Document Summary Agent
- Task Extraction Agent (approval required)
- Project Briefing Agent
- Recommendation cards
- Trust Dashboard

**Excludes:** External email sending, payments, marketplace, mobile native, advanced estimating

## CUSTOMER DISCOVERY SCRIPT

**Opening:**
"I'm testing Atlas as an AI project command center for contractors. Does it help organize project files, emails, tasks, and next actions in one trusted place?"

**Problem questions:**
- Where does your project information usually live?
- How often do tasks get buried in emails or documents?
- What part of project admin wastes the most time?

**Trust questions:**
- Would you connect Gmail or Drive to a tool like this?
- What would make that feel safe?
- Would you want to see exactly what the AI used?

**Pricing questions:**
- If this saved you 3-5 hours a week, what would it be worth?
- Would you pay monthly? Per project?
